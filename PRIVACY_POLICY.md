# Quick Sticky 개인정보처리방침

최종 업데이트: 2026-02-23

Quick Sticky(이하 "확장 프로그램")는 사용자 메모 기능 제공을 위해 최소한의 데이터만 처리합니다.

## 1. 수집 및 처리하는 정보

확장 프로그램은 다음 정보를 처리할 수 있습니다.

- 사용자가 직접 입력한 메모 텍스트
- 사용자가 메모에 드래그 앤 드롭한 이미지(Base64 형태)
- 사용자가 입력한 유튜브 링크로 생성된 카드 정보(제목, 썸네일 URL, 링크 URL)
- 메모 위치 좌표, 생성/수정 시각, 닫힘 상태 등 메모 동작에 필요한 메타데이터

## 2. 정보 처리 목적

위 정보는 아래 목적에만 사용됩니다.

- 웹페이지에서 메모 생성/수정/복원 기능 제공
- 브라우저 새로고침 또는 재방문 시 메모 상태 유지
- Note History(이력) 목록 제공 및 정렬/필터 기능 제공

## 3. 저장 위치 및 보관

- 데이터는 기본적으로 사용자의 브라우저 저장소(`chrome.storage.local`)에 저장됩니다.
- 별도 서버 데이터베이스로 전송하여 보관하지 않습니다.
- 사용자가 메모를 삭제하면 해당 메모 데이터는 로컬 저장소에서 제거됩니다.

## 4. 제3자 제공 및 외부 전송

- 확장 프로그램은 사용자 메모 데이터를 판매하거나 제3자에게 제공하지 않습니다.
- 단, 사용자가 유튜브 링크 기능을 사용할 경우 유튜브 oEmbed 조회 요청이 발생할 수 있습니다.
  - 이 요청은 유튜브 카드(제목/썸네일) 생성을 위한 목적입니다.

## 5. 권한 사용 안내

확장 프로그램은 다음 권한을 사용합니다.

- `storage`: 메모 데이터를 로컬에 저장/복원하기 위해 사용
- `tabs`: 단축키 명령 시 활성 탭에 메모 생성 신호를 전달하기 위해 사용
- `https://www.youtube.com/*` 호스트 권한: 유튜브 카드 정보 조회를 위해 사용

## 6. 사용자 선택권

사용자는 다음을 통해 데이터를 통제할 수 있습니다.

- 메모 `Close` 또는 `Delete` 기능 사용
- 브라우저에서 확장 프로그램 제거
- 브라우저의 확장 프로그램 데이터 삭제 기능 사용

## 7. 아동의 개인정보

본 확장 프로그램은 아동을 대상으로 설계되지 않았으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

## 8. 보안

확장 프로그램은 기능 수행에 필요한 최소 정보만 처리하며, 코드 내 입력 검증과 제한(예: 이미지 형식/크기 제한)을 적용합니다.
다만 사용자의 로컬 환경 보안 상태에 따라 데이터 보호 수준은 달라질 수 있습니다.

## 9. 개인정보처리방침 변경

본 방침은 기능 또는 법령 변경에 따라 수정될 수 있으며, 변경 시 업데이트 날짜를 통해 고지합니다.
